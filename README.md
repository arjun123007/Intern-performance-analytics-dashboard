# Intern Performance Analytics and Insights Dashboard

## Project Overview
This project analyzes intern performance data and provides insights through data preprocessing, analysis, and visualization. The goal is to understand factors affecting intern productivity, satisfaction, and overall performance.

## Dataset
The dataset contains information about interns including:
- Employee_ID
- Department
- Gender
- Age
- Job_Title
- Hire_Date
- Years_At_Company
- Education_Level
- Performance_Score
- Monthly_Stipend
- Work_Hours_Per_Week
- Projects_Handled
- Overtime_Hours
- Sick_Days
- Remote_Work_Frequency
- Team_Size
- Training_Hours
- Promotions
- Intern_Satisfaction_Score
- Resigned

## Project Steps

### 1. Data Collection
The dataset was collected from Kaggle and stored in CSV format.

### 2. Data Preprocessing
The following preprocessing steps were performed:
- Removed unnecessary columns
- Converted Hire_Date to datetime format
- Cleaned column names
- Checked for missing values
- Removed duplicates
- Renamed Employee_Satisfaction_Score to Intern_Satisfaction_Score

### 3. Data Analysis
The dataset is analyzed to identify patterns such as:
- Performance trends
- Department productivity
- Employee satisfaction
- Workload analysis

### 4. Dashboard
A dashboard will be created to visualize insights such as:
- Average performance score
- Department performance comparison
- Employee satisfaction distribution
- Work hours vs performance
